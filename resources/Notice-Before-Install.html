*Notice:

Please add the installation folder to PATH after setup, otherwise you won't be able to launch the program in different modes.


Few things to notice before we start:

1.There are 3 different versions of AdminCon  -  .NET Edition, Basic Edition and C++ Edition.

Edition: Filename: Architecture: Safe Mode: Script Mode: UI Mode:
.NET Edition AC.EXE .NET Framework 4.0 & .NET Core 2.1 Supported Supported Supported
Basic Edition ACBASIC.EXE .NET Framework 4.0 & .NET Core 2.1 Not Supported Not Supported Not Supported
C++ Edition ACWIN32.EXE Win32 (Visual C++ 2010 or higher is required) Not Supported Not Supported Not Supported

All of these editions of AdminCon will be installed.

2. Why I need to add the installation folder to PATH and how?

You can still use it by clicking the shortcut on desktop. However, you have to provide a command-line argument to start in different modes.
For examples:

ac -s: launch ac.exe in safe mode;

ac -ui: launch ac.exe in UI mode;

ac /?: show help of ac.exe.
